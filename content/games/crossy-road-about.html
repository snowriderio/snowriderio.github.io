Crossy Road is an endless arcade crossing game where the player controls a character moving forward across roads, rivers, and railways. Developed by Hipster Whale, the game is available on mobile platforms and can also be played online in a browser without installation.

Core Controls

Crossy Road uses simple, tap-based controls designed for quick input:

  • Tap / Click: Move one step forward
  • Swipe Left / Right or Arrow Keys: Move sideways
  • Arrow Down: Step backward (limited use)
  • Do not stay idle: The screen scrolls forward automatically.
  • If the player pauses too long, an eagle will remove the character, ending the run.

Main Gameplay Loop

Each step forward increases the score by one point. The environment generates randomly, mixing roads, rivers, railways, trees, and static obstacles. Vehicles and trains move at different speeds, requiring timing rather than memorization. Rivers can only be crossed by landing on floating logs or platforms.

Key Features

Endless Arcade Structure

There is no final stage or endpoint. Each run aims to surpass previous scores. Progress is measured only by distance traveled and coins collected.

Scoring System

  • +1 point for each forward step
  • Coins add progression value but do not directly increase score
  • Higher distance = higher leaderboard ranking

No Fixed Levels

Crossy Road uses procedural generation. Each run presents a different layout, preventing memorization and encouraging situational awareness.

Tips for Consistent Progress

  • Observe traffic patterns before moving forward
  • Use sideways movement to wait for safe gaps
  • Listen for train audio signals before crossing tracks
  • Avoid unnecessary coin risks when paths are unsafe
  • Tap near the bottom of the screen to keep vision clear
